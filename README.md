## Controls ‚å®Ô∏è
* `cw` - clockwise
* `ccw` - counter clockwise

### Game Settings
| Key                           | Description                           	|
|-------------------------------|---------------------------------------	|
| <kbd>P</kbd>                  | Pause / Unpause the game                |
| <kbd>G</kbd>                  | Toggle 3D grid                         	|
| <kbd>F</kbd>                  | Switch between Gouraud & Phong Shading  |
| <kbd>V</kbd>                  | Switch between Orthographic & Perspective View  |
<br/>
